Base: P17 Design Edition stabile.
Intervento: solo struttura tecnica PWA.
Grafica, disposizione, funzioni e logica meteo non sono state modificate.

CARICAMENTO SU GITHUB
1. Caricare tutti i file e la cartella icons nella radice del repository Radar-Conte.
2. Sostituire i file esistenti.
3. Attendere 1-2 minuti la pubblicazione di GitHub Pages.
4. Aprire in Chrome: https://conterb80.github.io/Radar-Conte/?v=18
5. Ricaricare la pagina una volta e attendere alcuni secondi.
6. Menu Chrome: dovrebbe comparire “Installa app”.

Se Chrome conserva la vecchia versione:
- chiudere tutte le schede Radar Conte;
- riaprire il link ?v=18;
- oppure cancellare solo i dati del sito conterb80.github.io e riaprire.
