CONTENUTO
- Mappa personalizzata centrata su Borgo Viazza
- Ultime scansioni radar animate
- Play/Pausa, avanti, indietro e ultimo fotogramma
- Zoom Locale e Romagna
- Indicazione dell'ora e dell'età del dato
- Controllo trasparenza
- Installazione PWA

INSTALLAZIONE SU GITHUB PAGES
1. Crea un nuovo repository, ad esempio Radar-Conte.
2. Carica nella radice tutti i file e la cartella icons contenuti in questo pacchetto.
3. Attiva GitHub Pages da Settings > Pages > Deploy from branch > main / root.
4. Apri l'indirizzo generato da GitHub Pages in Chrome.
5. Per installare: menu Chrome > Aggiungi a schermata Home / Installa app.

NOTA IMPORTANTE
Questa P1 usa RainViewer API per uso personale. Il servizio gratuito fornisce i fotogrammi radar passati e non l'evoluzione futura. Serve a verificare la fattibilità della nostra mappa personale prima di integrare dati ufficiali/nowcasting.
